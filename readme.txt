== Description ==
Bricks Portfolio is a Gutenberg block where you can display a portfolio list with different categories. A super-easy way to create a beautiful and responsive portfolio block for your WordPress websites.

It allows you to showcase your project's info in a clean layout. Minimal and sleek, you can showcase a project title, a short description, choose how many images display a single row on the gallery portfolio. 

Portfolio Block is fully responsive and works with all devices like Macbook, desktop PC, tablets, mobile. The first portfolio block plugin in WordPress for a separate category page.
